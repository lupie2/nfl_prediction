## NFL Match Prediction

Project, where I want to train a machine learning model to try and predict upcoming NFL matches like the Superbowl.
Data sourced from https://nflreadpy.nflverse.com/

1. Go through data and sort it by important variables
2. Clean it up
3. Train model
4. Create tkinter UI where user has 2 entry options to choose between teams and then get a probability of who the winner will be
